# LR1 PULLER - Neon Green Cyberpunk Edition

**Advanced Ome.tv IP Puller** with sleek neon green cyberpunk interface.

**There is 2 options the `Hungarian` one and the `English` one**

**If you are `Hungarian` use the hungarian one if you are `English` use the english one**

Automatically grabs the stranger's IP via WebRTC on ome.tv, shows location & ISP info, and silently sends the data + screenshot to your Discord webhook.

- Beautiful Neon Green Cyberpunk UI with Matrix rain background

## 🚀 How to Install and Use (Step by Step)

1. Install the **Tampermonkey** or **Violentmonkey** extension from the Chrome Web Store / Firefox Add-ons
2. After installation, click on the extension icon in your browser toolbar
3. Select **"Create a new script"** (or "Add a new script")
4. Delete all the default code that appears
5. Copy and paste the entire LR1 PULLER script code into the editor
6. **Configure your keys**:
   - Replace `e2bfc0786f8c00` with your own **ipinfo.io** API key
   - Replace the long Discord link with your own **Discord webhook URL**
7. Press **Ctrl + S** to save the script
8. Go to [https://ome.tv](https://ome.tv) and start chatting with random people
9. The neon green panel will automatically appear on the screen
10. When an IP is detected, the panel will show the information and silently send it to your Discord (no extra message on screen)

### Hotkeys (while on ome.tv):
- Press **`J`** → Save all captured IPs to a `.txt` file
- Press **`O`** → Take a manual screenshot
